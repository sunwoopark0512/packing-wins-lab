# Disclosure Snippets (Standardized)

All posts must satisfy: FTC (Relationship) + Amazon (Program) + Platform (Toggle/Label).

## 1. Short Snippets (Core 9)

**Placement**: Start of caption or very clearly at end (before hashtags).

- **KO**: #광고 #제휴 #협찬 (솔직한 리뷰)
- **EN**: #ad #sponsored #affiliate

## 2. Amazon Associates Preset

**Placement**: Anywhere in the post/description/bio where the link resides.

- **KO**: "이 포스팅은 쿠팡/아마존 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다."
- **EN**: "As an Amazon Associate I earn from qualifying purchases."

## 3. Blog Top Notice

**Placement**: Very top of the blog body.

- **KO**: "본 콘텐츠는 제휴마케팅 혜택이 포함되어 있으며, 솔직한 사용 경험을 바탕으로 작성되었습니다."
- **EN**: "This post contains affiliate links. Honestly written based on actual experience."

## 4. Platform Feature Checklist

If the platform has a native toggle, YOU MUST USE IT (in addition to text).

- [ ] **TikTok**: Toggle "Commercial Content" -> "Detailed Disclosure" -> "Your Brand" (ON)
- [ ] **YouTube Shorts**: Check "Includes paid promotion" (ON)
- [ ] **Instagram**: Use "Paid Partnership" label (ON)
- [ ] **Facebook**: Tag "Brand Partner" (ON)

## 5. Rules

- **One Link Rule**: Use `https://YOUR-LINK-HUB.example` (LinkHub) only. No direct affiliate links in caption if avoid spam.
- **Truth Rule**: Never say "I used this" if you didn't. Used "Recommended for..." instead.
