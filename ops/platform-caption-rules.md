# Platform Caption Rules (9 platforms)

목표: 같은 원천이라도 플랫폼별로 ‘첫 문장/길이/CTA’를 다르게 해서 중복 감지/피로도를 줄인다.

## 9 플랫폼 기준(필요 시 교체 가능)

1. TikTok
2. YouTube Shorts
3. Instagram Reels
4. Facebook Reels
5. Threads
6. X(Twitter)
7. Pinterest Idea Pins
8. LinkedIn
9. Blog(WordPress/Medium 중 1)

## 캡션 규칙(공통)

- 1문장 훅(10~14자) → 핵심 이득 1개 → CTA 1개
- 금지: 과장 수치/확인 불가 사실 단정/정책 위반 유도
- 해시태그: 3~6개(플랫폼별 다름)

## 플랫폼별 차등

- TikTok: 질문형 훅 + 댓글 유도
- Shorts: “결론 먼저” + 고정댓글 CTA
- Reels: 감정 단어 1개 + 저장 유도
- Facebook: 공감형 문장 + 공유 유도
- Threads: 짧은 연속 2~3줄 + 대화 유도
- X: 한 줄 인사이트 + 3줄 스레드 템플릿
- Pinterest: 검색 키워드 중심 + To-do 느낌
- LinkedIn: 문제→배운점→프레임워크(짧게) + 신뢰 톤
- Blog: SEO H2 3개 + FAQ 3개 + CTA 1개
